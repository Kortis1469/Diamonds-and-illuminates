# Diamonds and illuminates
Это мой первый проект с использование SDL.
Можно считать это квазе игровым движком. Грубо говоря это тестовая сцена.
В этом "движке" реализованы кривые Безье, и такие операции над фигурами как: вращение,
линейное перемещение и скейлинг.

https://github.com/user-attachments/assets/0ca7a747-7ac6-456b-8326-896643f0c9ec

Треугольник зовут Илья. Полное имя Иллюминат.
Илья отрендерен полностью на "движке".

## Futures:
В дальшеем планируется перевести его на новую версию "движка", более доработанную и
добавить несколько сцен, где Илью, к примеру, можно будет таскать по экрану, изменять
его упругость и смотреть как он впечатывается в стены.

Также будет добавлены антиаллиасинг, физика частиц и другие мелкие, но важные детали.
